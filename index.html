Developing Site-Spesific Management for Drone Sprayer

üìÖ 2023 ‚Äì 2024

Tools: QGIS, SQL, GDAL, Rasterio, Fiona, Arduino IDE, DJI Agras T16, DJI Mavic 3 Multipectral

A Variable Rate Application (VRA) system was developed for the DJI Agras T16 drone sprayer to enhance spraying accuracy and support sustainable agriculture. This system utilizes dynamic pump flow control powered by Raspberry Pi and Arduino, enabling precise application of inputs. Python libraries are employed to process multispectral mapping data into GeoTIFF treatment maps for map-based spray adjustments.

GDAL handles geospatial data transformation, Rasterio extracts raster-based treatment values, and Fiona converts these into vector polygons for variable rate spraying. This innovation reduces input waste and environmental impact. Later, I advanced the system using the DJI Agras T40, incorporating adjustable flow rates and droplet sizes for even greater efficiency.

Machine Learning for Land Use and Land Cover (LULC) Analysis

üìÖ2024

Tools QGIS, Semi-Automatic Classification Plugin, Orfeo Toolbox, Google Earth Engine

Satellite imagery from Google Earth Engine (GEE) and EO Browser was analyzed using supervised and unsupervised machine learning techniques to classify Land Use and Land Cover (LULC). Preprocessing and feature extraction were performed using Orfeo Toolbox and the Semi-Automatic Classification Plugin (SCP) in QGIS. Algorithms such as Decision Trees (DT), Support Vector Machines (SVM), Random Forest (RF), and K-Means Clustering were applied to achieve accurate classifications.

This project highlights expertise in geospatial data processing and the application of machine learning for remote sensing. By leveraging advanced tools and techniques, the workflow ensures reliable LULC classification, supporting applications in environmental monitoring, land management, and spatial planning.

Administrative Boundary Polygons of Indonesia Down to the Village Level

üìÖ 2021

Tools: SQL, QGIS, GeoJSON, VS Code, JavaScript, Leaflet, CSS

A web-based system was created to visualize Indonesia's administrative boundaries down to the village level using official data from the Ministry of Home Affairs across 38 provinces. Extensive polygon datasets were processed with SQL for database management and QGIS for geospatial preparation. Simplified GeoJSON files ensured fast web rendering without compromising spatial accuracy. The system was built using JavaScript, and the Leaflet library for interactive maps, with CSS for responsive design.

This project showcases expertise in geospatial data processing, optimization for web performance, and interactive visualization. By combining robust tools like VS Code, QGIS, and modern web technologies, it delivers a scalable and efficient platform for accurate geospatial information, supporting public administration, research, and development needs.
